# Systematische Messabweichung
Zwei unbekannte Widerstände $R$ im $\mathrm{k\Omega}$-Bereich werden mit zwei Messschaltungen bestimmt:

```{figure} pictures/strom_spannungsrichtig.png
:class: .dark-light
---
height: 150px
name: optional-label
---
Schaltungen zur Bestimmen von Widerständen nach $R = U/I$

```

* Welches Schaltbild stellt die stromrichtige und welche die spannungsrichtige Schaltung dar?
* Berechnen Sie für beide Fälle den unkorrigierten und korrigierten Widerstandswert. Bei welcher Schaltung ist der relative Fehler kleiner?
* Bestimmen Sie für einen Strom von $I = 140\mathrm{mA}$, einer Spannung von $U=10\mathrm V$ und den Innenwiderständen $R_{iA}=4\Omega$ und $R_{iV}= 10\mathrm{k}\Omega$ die bekannte systematische Messabweichung der Leistung $P$ über $R$ mittels Schaltung 2. Geben Sie das korrigierte Messergebnis der Leistung an. 


````{admonition} Tips
:class: dropdown
* Schaltung 1 (korrigiert):
$$U = RI = R\cdot (I-I_V)$$
$$I_V = U / R_{iV}$$
$$\Rightarrow R = ... $$

* Schaltung 2 (korrigiert):
$$U = RI $$
$$R = R_{iA} + R$$
$$\Rightarrow R = ... $$

* Leistungsmessung in Schaltung 2 (korrigiert):
$$P = UI $$
$$U = I\cdot (R_{iA} + R)$$
$$P = ... $$
````
