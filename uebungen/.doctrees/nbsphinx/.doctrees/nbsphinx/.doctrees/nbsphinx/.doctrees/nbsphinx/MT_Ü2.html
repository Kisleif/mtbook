√ú2.1 Dampfdruckthermometer #

Der Dampfdruck, \(p_\mathrm D\), ist der Druck, der sich einstellt, wenn sich in einem abgeschlossenen System ein Dampf mit der zugeh√∂rigen fl√ºssigen Phase im thermodynamischen Gleichgewicht befindet. F√ºr verschiedene Temperaturen folgt die Dampfdruckkurve der Exponentialfunktion

\[ p_\mathrm{D} = c \cdot \mathrm{e}^{-\frac{\Delta E}{kT}}\]

wobei \(c\) eine Konstante ist, \(k = 1,380649\cdot 10^{-23}\,\mathrm{J/K}\) die Boltzmann Konstante und \(\Delta E\) die Verdampfungsenergie eines Molek√ºls.

# Definition der Exp-Funktion in Python-Code
k = 1.380649e-23 # in J/K
def pD(c,dE,T):
    return c * np.exp(-dE/(k*T)) # in bar

Durch Logarithmieren beider Seiten erh√§lt man die √§quivalente logarithmische Darstellung

\[\ln{p_\mathrm{D}} = A - B/T\]

wobei \(A\) und \(B\) wieder die Materialkonstanten sind. durch Vergleichen der exponentielle und logarithmischen Schreibweise k√∂nnen die Materialkonstanten ineinander umgerechnet werden:

\[c = \mathrm e^{A}\]
\[\Delta E = B\cdot k\]
# Definition von Funktionen um die Materialkonstanten ineinander umzurechnen
def calc_c(A):
    return np.exp(A)
def calc_dE(B):
    return B*k

Empfindlichkeit des Dampfdruckthermometers f√ºr Wasser #

Um die Empfindlichkeit anhand der Kennlinie zu bestimmen, m√ºssen wir die Dampfdruckgleichung von oben ableiten:

# Funktion um pD abzuleiten. 
# Es werden einfach zwei aufeinanderfolgende Funktionswerte analysiert und die Steigung berechnet
def d_pD(c,dE,T): # in bar/K
    h = 1e-5 # in der Theorie w√§re h infinitesimal klein (Abstand zwischen zwei Temperaturen)
    return (pD(c,dE,T+h)-pD(c,dE,T))/h

√úbergang zum Gasthermometer #

Das Wasser-Dampfdruckthermometer besteht aus einen Gef√§√ü mit einem Volumen von 1 Liter und einer Druckfestigkeit von 50 bar. Aus Sicherheitsgr√ºnden wird beim Bau des Thermometers nur soviel Fl√ºssigkeit eingef√ºllt, dass bei einer Temperatur von 250¬∞C die ganze Fl√ºssigkeit verdampft ist. Dies entspricht fast der Avogadro-Konstanten \(N_A = 6.022\cdot 10^{23}/\,\mathrm{mol}\), welche die Anzahl von Molek√ºlen in 1 mol definiert.

√ú2.2 Systematische Messabweichung #

Messger√§te k√∂nnen die genaue Messung von elektrischen Gr√∂√üen beeinflussen. Je nachdem, ob die Stromst√§rke I oder die Spannung U besonders genau gemessen werden soll, m√ºssen die Messger√§te entsprechend geschaltet sein.

Schaltung 1: Spannungsrichtige Schaltung #

M√∂chte man einen systematischen Fehler bei der Spannungsmessung vermeiden, so w√§hlt man die spannungsrichtige Schaltung. Bei der spannungsrichtigen Messung wird nur die √ºber dem Bauteil (Widerstand R) abfallende Spannung UR gemessen. Da kein weiteres Bauteil die Messung beeinflusst, wird hier die Spannung korrekt gemessen.

Image("schaltung1.png")

Der unkorrigierte Widerstandswert berechnet sich aus

\[ R = \frac{U}{I}\]
U = 10
I = 2.4e-3
RiA = 4
RiV = 10e3
R = U/I # unkorrigierter Widerstandswert
print('Der unkorrigierte Widerstandswert ist R =', R, 'Ohm')

Der unkorrigierte Widerstandswert ist R = 4166.666666666667 Ohm

Allerdings registriert das Strommessger√§t in dieser Schaltung nicht nur den Strom IR, der durch den Widerstand flie√üt, sondern auch den durch das Messger√§t flie√üenden Strom IV:

\[I_V = \frac{U}{R_{iV}}\]
IV = U/RiV # Strom, welcher √ºber den Spannungsmesser abflie√üt
print('Der Strom der durch das Spannungsmessger√§t flie√üt ist IV =', IV, 'A')

Der Strom der durch das Spannungsmessger√§t flie√üt ist IV = 0.001 A

Die Strommessung ist in dieser Schaltung mit einem systematischen Fehler behaftet. Die Strommessung muss entsprechend korrigiert werden:

\[I_R = I - I_V\]
IR = I - IV # korrigierter Stromwert, welcher √ºber R abflie√üt
print('Korrektur der Strommessung liefert IR =', IR, 'A')

Korrektur der Strommessung liefert IR = 0.0013999999999999998 A

Dadurch lann der korrigierte Widerstandswert bestimmt werden:

\[ R_k = U / I_R \]
Rk = U/IR # korrigierter Widerstandswert
print('Der korrigierte Widerstandswert lautet R =', Rk, 'Ohm')

Der korrigierte Widerstandswert lautet R = 7142.857142857144 Ohm

Der absolute Fehler der Widerstandsmessung betr√§gt somit

\[A = R - R_k = \Delta R\]

undf√ºr den relativen Fehler bezogen auf den Messwert folgt:

\[A_\mathrm{rel} = A/R_k.\]
A = R - Rk # absoluter systematischer Fehler
Arel = A/Rk * 100 # relativer systematischer Fehler
print('Der absolute Fehler betr√§gt: ', A, 'Ohm')
print('Der relative Fehler betr√§gt: ', Arel, '%')

Der absolute Fehler betr√§gt:  -2976.190476190477 Ohm
Der relative Fehler betr√§gt:  -41.66666666666667 %

Der Fehler kann durch einen hohen Eingangswiderstand des Messger√§tes von einigen MŒ© gering gehalten werden. Dadurch wird auch der durch den Spannungsmesser flie√üende Strom klein.

Schaltung 2: Stromrichtige Schaltung #

Bei der stromrichtigen Messung wird nur der durch ein Bauteil (Widerstand R) flie√üende Strom gemessen. Dazu wird der Strom ohne Verzweigung unmittelbar vor oder hinter dem Bauteil gemessen. In dieser Schaltung wird die Stromst√§rke IR korrekt gemessen.

Das Spannungsmessger√§t registriert in dieser Schaltung nicht nur die an R abfallende Spannung UR, sondern auch die √ºber dem Strommessger√§t abfallende Spannung UA:

\[U_A = R_{iA} \cdot I\]
UA = RiA*I # Spannungsabfall am Strommessger√§t
print('Die Spannung, die √ºber dem Strommessger√§t abf√§llt ist UA =', UA, 'V')

Die Spannung, die √ºber dem Strommessger√§t abf√§llt ist UA = 0.0056 V

Die Spannungsmessung ist also in dieser Schaltung mit einem systematischen Fehler behaftet, welcher vom eigentlichen Spannungswert abgezogen werden muss:

\[U_R = U - U_A.\]

print('Korrektur der Spannungsmessung liefert UR =', UR, 'V')
