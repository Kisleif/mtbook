Messdaten skalieren und UI-Kennline zeichnen#

Nun konvertieren die Messdaten gem√§√ü der Versuchsanleitung in die richtigen Gr√∂√üen um:

\[I_{\mathrm{diode}} = \frac{U_\mathrm{ch1}}{R_{2}} = \frac{U_\mathrm{R_{2}}}{R_{2}}\]
\[U_\mathrm{diode} = U_\mathrm{ch2} - U_\mathrm{R_{2}} = U_\mathrm{ch2} - U_\mathrm{ch1}\]

(Kontrolliert unbedingt ob ihr wirklich \(R_2 = 100 \Omega\) eingebaut hattet! Ansonten m√ºsst ihr den Wert hier entsprechend √§ndern.)

R2 = 100
data1_cut['I_diode'] = data1_cut['Ch1_Volt']/R2
data1_cut['U_diode'] = data1_cut['Ch2_Volt'] - data1_cut['Ch1_Volt']

Funktion f√ºr die Ideale Kennlinie#

Die Messdaten sollen nun mit der Modellfunktion der idealen Kennlinie gem√§√ü dem Anleitungsskript gezeichnet werden:

\[I(U) = I_{R}(U) + I_{G} = I_{R}(U) - I_{R}(0) = I_{R}(0)\cdot \exp\left(\frac{eU}{k_{B}T}-1 \right)\]

Hierf√ºr definieren wir zun√§chst die Funktion in Python, \(I(U)\), mit den entsprechenden Naturkonstanten:

def I_U(U):
    T = 300 # als Temperatur w√§hlen wir 300K, was in etwa Raumtemperatur entspricht. 
    k = 1.3e-23 # Boltzmann-Konstante in J/K
    e = 1.602e-19 # Elementarladung in C
    I_0 = 1.2e-14 # Stromst√§rke bei U = 0V
    return I_0 * np.exp(e*U/(k*T) -1)

Messdaten und Funktion werden in einem Diagramm gezeichnet. Um die Funktion darzustellen nutzen wir die gemessenen Spannungsdaten als x-Achse. Alternativ h√§tten wir auch ein neues Array erstellen k√∂nnen.

plt.plot(data1_cut['U_diode'], I_U(data1_cut['U_diode']), label = 'Ideale Kennlinie, 300K', color = 'tab:orange')
plt.xlabel('Spannung U (V)')
plt.ylabel('Stromst√§rke I (A)')

plt.show()

Funktion an Messdaten anpassen#

Zum fitten benutzen wir die curve_fit Funktion von scipy.

from scipy.optimize import curve_fit

Wir fitten in logerithmischen Einheiten auf der Y-Achse und rechnen daher die Stromst√§rke in die Einheit dB um und speichern sie in der Variablen y. Die Spannungsdaten bleiben unver√§ndert, werden nur in x abgespeichert.

x = data1_cut['U_Volt'].dropna()
y = 10*np.log10(data1_cut['I_Ampere'].dropna())

plt.figure(figsize=(8,5)) # Diagramm-Gr√∂√üe einstellen
plt.plot(x,y, 'o', label = 'gemessen Kennlinie der Diode XY')
plt.xlabel('Spannung U (V)')
plt.ylabel('Stromst√§rke I (dB)')
plt.legend()
plt.show()

Das Diagramm zeigt eine identische Repr√§sentation der Messdaten wie im vorangegangenen Abschnitt mit der logarithmischen Y-Achse. Nun wurden jedoch die Y-Messwerte anstelle der Achse umgerechnet und wir k√∂nnen die Messdaten mit einer einfachen linearen Regression anpassen (ansonsten h√§tten wir eine exponentielle Fit-Funktion ben√∂tigt). Zun√§chst definieren wie hierf√ºr die lineare Funktion

\[y = ax + b\]
def fit_func(x, a, b):
    return a * x + b

Dann fitten wir die Funktion an die Messdaten an:

popt, pcov = curve_fit(fit_func, x , y)

Anschlie√üend zeichnen wir das Ergebnis gemeinsam mit den Messwerten in ein Diagramm und geben uns das Fit-Ergebnis aus:

plt.figure(figsize=(8,5)) # Diagramm-Gr√∂√üe einstellen
plt.plot(x,y, 'o', label = 'gemessen Kennlinie der Diode XY')
plt.plot(x,fit_func(x,*popt), label = 'Modellfunktion')
plt.xlabel('Spannung U (V)')
plt.ylabel('Stromst√§rke I (dB)')
plt.legend()
plt.show()

print('Y-Achsenabschnitt f√ºr y = U = 0 ist: ', popt[1], 'dB = ',  10**(popt[1]/10), 'A')

Y-Achsenabschnitt f√ºr U = 0 ist:  -87.263994734694 dB =  1.8775889752721304e-09 A
